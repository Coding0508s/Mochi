# Store 재고 품목 관리 가이드

스토어 재고 화면에 노출할 품목코드를 관리자 화면에서 운영하는 방법입니다.

## 1) 화면 위치

- 좌측 메뉴: `CO Team > Store 재고품목`
- URL: `/store/inventory/skus`
- 보조 진입: `CO Team > Store 재고` 화면의 `품목 관리` 버튼(모달)
- 권한: `manageStoreInventory` (관리자만 접근 가능)

## 2) 기본 규칙

- 입력값은 이카운트 품목코드(`PROD_CD`) 기준입니다.
- `활성` 상태인 품목만 Store 재고 조회 대상이 됩니다.
- 목록·재고 조회에 쓰이는 품목 순서는 **품목코드(`PROD_CD`) 문자 오름차순**입니다 (DB `sort_order`와 무관).

## 3) 단건 등록

1. `연동 품목 추가` 영역에 품목코드 입력
2. 필요한 경우 메모 입력
3. `품목 추가` 클릭

## 4) 일괄 등록 (초기 세팅)

1. `일괄 등록` 영역에 품목코드를 붙여넣기
2. 구분자는 쉼표(`,`) 또는 줄바꿈 사용 가능
3. `일괄 등록` 클릭
4. 결과 메시지에서 `추가 N건 / 중복 N건` 확인

예시 입력:

```text
00P228,00P227,00P211
00P017R
VGCHQ01
```

## 5) 운영 팁

- 품목 추가 후 Store 재고 화면에서 검색으로 즉시 확인하세요.
- 비활성화된 품목은 API 조회 대상에서 제외됩니다.
- 대량 변경 전에는 활성 목록을 먼저 백업(복사)해 두는 것을 권장합니다.
- `최근 차감` 정보는 이카운트 **수불 이력**과(선택) **판매 목록(GetListSale)** 조회 결과를 합쳐, 품목별로 더 최근 시각 기준으로 표시됩니다.
- 차감구분 라벨은 `주문연동 / 자가사용 / 재고조정 / 기타차감`으로 표기됩니다.
