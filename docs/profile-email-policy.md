# Profile Email Policy

## 목적
- 프로필 화면의 이메일 입력을 브라우저 조작으로 우회 변경하는 시도를 차단합니다.
- 사용자가 \"저장이 안 된 이유\"를 즉시 이해할 수 있도록 명시적인 오류 메시지를 제공합니다.

## 정책
- `GET /profile` 화면에서 이메일은 읽기 전용입니다.
- `PATCH /profile` 요청에서 이메일은 **현재 로그인 계정 이메일과 완전히 동일**해야 합니다.
- 이메일 값이 변경되거나 누락되면 저장은 실패하며, 정책 안내 메시지를 반환합니다.

## 사용자 안내 문구
- 기본 안내: `이메일은 프로필에서 변경할 수 없습니다. 변경이 필요하면 관리자에게 요청해 주세요.`
- 누락 요청 안내: `이메일은 프로필에서 변경할 수 없습니다. 화면을 새로고침한 뒤 다시 시도해 주세요.`

## 운영 요청 경로
- 이메일 변경이 필요한 경우, 사용자 프로필이 아닌 운영 권한자에게 변경을 요청합니다.
- 운영 권한자는 계정 정보와 직원 마스터 정합성을 함께 확인한 뒤 처리합니다.
